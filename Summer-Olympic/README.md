# Summer Olympic Games Analysis

This project analyzes historical data from the Summer Olympic Games
and demonstrates the step-by-step evolution of a Tableau dashboard.

The analysis was developed iteratively in multiple stages, with each stage
adding deeper insights, improved structure, and more refined data storytelling.

**Tool:** Tableau Desktop  
**Author:** Mahnoosh Shoaibi

---

## Project Timeline & Dashboard Evolution

- **Stage 1 – January 12, 2025:**  
  Initial overview focusing on medal distribution, countries, and Olympic host cities.

- **Stage 2 – January 25, 2025:**  
  Extended analysis with athlete participation trends and gender representation.

- **Stage 3 – February 6, 2025:**  
  Final refinement including host city insights, sport category analysis,
  and improved interactivity.
